# Screenshots Directory

Place your UI screenshots here.

## Recommended Screenshots

1. **claims-prediction-dashboard.png** - Main ML prediction dashboard
   - Shows the file upload interface
   - Displays prediction results table
   - Shows summary statistics

2. **reports-list.png** (optional) - Reports listing page
   - Shows generated reports
   - Download links

3. **api-docs.png** (optional) - FastAPI interactive documentation
   - Shows the `/docs` endpoint

## Screenshot Guidelines

- **Format**: PNG or JPG
- **Size**: Recommended 1920x1080 or similar
- **Naming**: Use kebab-case (e.g., `claims-prediction-dashboard.png`)
- **Quality**: High resolution for clarity

## How to Take Screenshots

1. Start both backend and frontend servers
2. Navigate to `http://localhost:3000`
3. Upload a sample file and get predictions
4. Take a screenshot of the results
5. Save it in this directory with the name referenced in README.md
